# VSCode Task Runner

Implemented features:

- [x] Predefined variables:
  - `${userHome}`
  - `${workspaceFolder}`
  - `${workspaceFolderBasename}`
  - `${pathSeparator}`
  - `${env:VARIABLE}`
- [x] Settings hierarchy:
  - Global level settings
  - Global level OS-specific settings
  - Task level settings
  - Task level OS-specific settings
- [x] Task configuration:
  - `cwd`
  - `env`
  - `type`
  - `command`
  - `args`
